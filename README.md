# k9-punsj-frontend

Frontend for manuell `punching` av papirsøknader som kommer inn for ytelser i kapittel 9.

## Komme i gang
For å kjøre opp applikasjonen:
````
npm install
npm start
````
Følgende kommando vil kjøre opp testdata generert av Jsonserver:
````
npm run mock:api
````
